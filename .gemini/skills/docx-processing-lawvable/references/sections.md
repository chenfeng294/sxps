# Sections

| Command | Description | Key flags |
|---------|-------------|-----------|
| `sections list` | List all sections | (none) |
| `sections get` | Get section info | `--section-id` or address |
| `sections set-break-type` | Set section break type | `--target-json`, `--break-type`, `--dry-run` |
| `sections set-page-margins` | Set page margins | `--target-json`, `--top`, `--bottom`, `--left`, `--right`, `--dry-run` |
| `sections set-header-footer-margins` | Set header/footer margins | `--target-json`, `--header`, `--footer`, `--dry-run` |
| `sections set-page-setup` | Set page size/orientation | `--target-json`, `--width`, `--height`, `--orientation`, `--dry-run` |
| `sections set-columns` | Set column configuration | `--target-json`, `--dry-run` |
| `sections set-line-numbering` | Enable line numbering | `--target-json`, `--dry-run` |
| `sections set-page-numbering` | Set page number format/start | `--target-json`, `--dry-run` |
| `sections set-title-page` | Enable/disable title page | `--target-json`, `--enabled`, `--dry-run` |
| `sections set-odd-even-headers-footers` | Enable odd/even mode | `--enabled`, `--dry-run` |
| `sections set-vertical-align` | Set vertical page alignment | `--target-json`, `--dry-run` |
| `sections set-section-direction` | Set text flow direction | `--target-json`, `--dry-run` |
| `sections set-header-footer-ref` | Set header/footer reference | `--target-json`, `--dry-run` |
| `sections clear-header-footer-ref` | Clear header/footer reference | `--target-json`, `--dry-run` |
| `sections set-link-to-previous` | Link/unlink to previous section | `--target-json`, `--dry-run` |
| `sections set-page-borders` | Set page borders | `--target-json`, `--dry-run` |
| `sections clear-page-borders` | Remove page borders | `--target-json`, `--dry-run` |
